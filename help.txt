emote add         (generate a new local artifact into an existing project)
emote create      (generate a new development structure) 
emote deploy      (upload a local (project) artifact to remote server)
emote download    (download a remote (server) artifact to a local project)
emote exec        (run a local directive against a remote server)
emote log         (show remote (server) log messages, tenant 'tail')
emote list        (show artifacts)
emote status      (show status of remote server)
emote undeploy    (remove/deactivate/clean a remote resource)
